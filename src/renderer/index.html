      Install Game Runtimes
      
        
      
      
      
      
        
      Some Microsoft runtime components that many of these games rely on are missing from your PC. Installing them now prevents VCRUNTIME140.dll, MSVCR120.dll and DirectX errors when you launch games. These are the official Microsoft redistributables.
      
        
        
        
        
          
        
          
        
        
        
        
        A Windows security prompt (UAC) will appear for each component — click Yes to allow the install.
